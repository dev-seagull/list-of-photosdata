GOOGLE PHOTOS API

These are the information that we can get from media items using Google Photos API:



Media items table

Description

Sample

Id Identifier for the media item
Description Description of the media item
Product Url Google Photos URL for the media item.
Base Url A URL to the media item's bytes
Mime Type MIME type of the media item
Meta data Metadata related to the media item, such as, height, width, or creation time
File name Filename of the media item

The meta data for a photo might include: cameraMake - cameraModel - focalLength - apertureFNumber - isoEquivalent - exposureTime

The meta data for a video might include: cameraMake - cameraModel - fps - status











These are the information that we can get from an album using Google Photos API:



Albums list table

Description

Sample

Id Identifier for the album
Title Name of the album
Product Url Google Photos URL for the album
Media items count The number of media items in the album
Cover photo BaseUrl A URL to the cover photo's bytes
Cover photo media item id Identifier for the media item associated with the cover photo.


Listing album contents: To list all of the media items in an album, add the albumId field to your search request. It returns the list of media items in the particular album.
